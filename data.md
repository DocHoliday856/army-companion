# Layout of XML

## Unit Info

- categoryEntries
  - categoryEntry
    - Mapping of troop types
      - HQ, Elites, Troops, etc.
    - mapping of unit types
      - id, name
- rules
  - rules for DG specific
- sharedSelectionEntries
  - selectionEntry
    - stats on units
- sharedProfiles
  - weapons that can be used by multiple units
